# Ticket Booking System

This is a full-stack ticket booking application built with a Node.js/Express backend and a React/TypeScript frontend. It features a robust, concurrency-safe API for booking seats for different shows.

## Live Application

- **Frontend (Vercel)**: [https://ticket-booking-system-modex.vercel.app/](https://ticket-booking-system-modex.vercel.app/)
- **Backend (Render)**: [https://ticketbookingsystem-modex.onrender.com](https://ticketbookingsystem-modex.onrender.com)

## Project Structure

This project is a monorepo containing both the backend and frontend code.

```
.
├── ticket_backend/         # Node.js/Express backend
│   └── backendreadme.md    # Backend-specific documentation
└── ticketBooking-frontend/ # React/TypeScript frontend

```
